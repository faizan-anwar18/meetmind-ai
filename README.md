# MeetMind AI

An AI-powered meeting assistant that converts meeting recordings into transcripts, generates intelligent summaries, extracts action items, and helps teams stay organized.

## 🚀 Tech Stack

- Frontend: React + Vite
- Backend: FastAPI
- Database: PostgreSQL
- AI: OpenAI / Gemini
- Speech-to-Text: Whisper
- Authentication: JWT
